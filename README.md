HENI
====

## Getting Started

This has been tested using Node v14.7.3 on MacOS.

1. Install dependencies:
    ```
    npm install
    ```

2. Run the unit tests:
    ```
    npm run test
    ```

3. Start the server and call the API:
    ```
    npm run start
    ```
    and in a separate terminal:
    ```
    curl http://localhost:3000/
    ```
